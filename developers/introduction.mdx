## Welcome to Polymarket Developer Documentation

Build powerful applications on top of Polymarket's prediction market infrastructure. Our developer tools provide comprehensive access to market data, trading capabilities, and real-time information streams.

## What You Can Build

With Polymarket's developer tools, you can:

- **Trading Bots:** Automated trading strategies and market-making bots
- **Analytics Platforms:** Market analysis and data visualization tools
- **Aggregators:** Multi-market displays and comparison tools
- **Integrations:** Embed markets in your application
- **Data Services:** Historical and real-time market data feeds
- **Custom Interfaces:** Alternative trading interfaces and experiences

## API Features

### Central Limit Order Book (CLOB)

- Place and manage orders
- Access order book data
- View market liquidity
- Execute trades programmatically

### Market Data

- Current market prices
- Historical price data
- Trade history
- Market metadata

### Real-Time Streams

- WebSocket connections
- Live price updates
- Order book changes
- Trade notifications

### Authentication

Most endpoints require authentication to access. You'll need to generate API credentials and sign your requests.

## Developer Support

### Resources

- **Documentation:** Comprehensive API reference and guides

### Rate Limits

API requests are subject to rate limits to ensure platform stability. Check the quickstart guide for current limits and best practices.

## Getting Started

Ready to start building? Head over to the [Quickstart Guide](/developers/quickstart) to make your first API call.

## Important Notes

- All trading is conducted in USDC on the Polygon network
- Markets are settled using UMA's Optimistic Oracle
- Smart contracts are auditable on the blockchain
- API access is provided as-is for development purposes
